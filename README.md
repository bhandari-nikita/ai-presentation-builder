# AI Presentation Builder

AI-powered presentation builder inspired by Gamma.

This project focuses on generating beautiful, structured presentations using reusable slide components, dynamic layouts, and AI-generated content.

---

# Features

## Current Features
- Next.js frontend setup
- Tailwind CSS UI
- Reusable component architecture
- Dynamic slide rendering
- Multiple slide navigation
- Presentation container system

---

# Planned Features
